# 芯片选型

接了一个很有挑战的需求，需要我把 0.5V 的电压转换到 2.0V。

传统的 Boost-Buck 电路无法实现这个需求，因为一个普通二极管的压降约为 0.7V，可是电源只能提供 0.5V 的电压。

大部分现有的电压转换芯片，需要 0.9V 的启动电压，电源不能满足这个要求。

找了很久，发现了 TPS6200 这款芯片，它支持 0.3V-5.5V 的输入，并且以 1.8V - 5.5V 的电压输出（可调且固定）

# PCB 设计

很好的 JYF 老师，指点我很多，使我的 PCB 旋转。

一开始，我参考了 Supplying TPS61200 With a Single Solar Cell 和 Using TPS61200 as WLED Driver 作出第一版设计，拿给 JYF 老师看：

没什么问题，就是有一些问题要改一下

—— JYF

然后 JYF 老师教了我很多关于电源电路设计的技巧，比如：

  • 铺铜时使用“阻止区”避免产生“天线”
  • 在“天线”的附近，打过孔，让干扰回流到地
  • 画板子，把板子尺寸设置成 5mm 倍数，好做外壳
  • 钢网在淘宝做更便宜
  • 横着的座子不好用，换成罚站的好用
  • …

虽然 JYF 老师偶尔“内涵”我一下，但还是得感谢 JYF 老师，他的帮助这是让这个项目一次成功的关键。

（顺便一提，我偷懒去掉了 MPP 电路，可能持续 20ms 对电源索取 1A 的电流，在某些燃料电池里相当于短路，从而影响电源寿命）

# 焊接

这是我第一次手工焊接 QFN 芯片，弄坏了一个 PCB 才焊接好。最后的成品质量很高。

之前焊接 0805 很丑，现在焊接 0603 封装现在也不在话下。以后失业可以去工厂打螺丝了
