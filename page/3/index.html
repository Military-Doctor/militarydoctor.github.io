RFC5389 - Session Traversal Utilities for NAT 翻译

STUN 是一种协议，用于检查两个终端之间的连通性，可以工作在多种 NAT 之中，不需要 NAT 对其有任何行为。

STUN 本身不是一个 NAT 穿透的解决方案，而是一个 NAT 穿透条件检测工具。
