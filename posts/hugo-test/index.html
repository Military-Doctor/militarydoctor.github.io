Hey, I want to debug linux kernel, what should I do?

Well, in short, we can debug the kernel:

  1. Boot a kernel in a virtual machine (QEMU).
  2. Attach GDB to QEMU

Sounds simple? What’s the challenge in these two steps?

  1. Boot a kernel in a virtual machine (QEMU).
    1. Compile QEMU (you can use binary release and skip this step)
    2. Compile kernel
      1. Config kernel to enable debugging
    3. Identify compiled kernel (bzImage, vmlinux.a, vmlinux.o)
    4. Making rootfs
    5. Booting a kernel by QEMU
      1. Setting proper arguments (disbale KASLR, )
  2. Attach GDB to QEMU
    1. Setting up your GDB
    2. Attach GDB to QEMU
    3. How to set breakpoints (GDB command refresher)
